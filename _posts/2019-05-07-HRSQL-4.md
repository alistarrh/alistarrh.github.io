# Select By ID

Query all columns for a city in **CITY** with the ID 1661.


**Input Format**

The **CITY** table is described as follows:

 <center>CITY</center>

|Field|Type|
|---|---|
|ID|NUMBER|
|NAME|VARCHAR2(17)|
|COUNTRYCODE|VARCHAR2(3)|
|DISTRICT|VARCHAR2(20)|
|POPULATION|NUMBER|

**Solution**--MySQL

```mysql
SELECT * FROM CITY
WHERE ID = 1661
```
