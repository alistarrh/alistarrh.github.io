# Weather Observation Station 1

Query a list of CITY and STATE from the **STATION** table.


**Input Format**

The **CITY** table is described as follows:

 <center>STATION</center>

|Field|Type|
|---|---|
|ID|NUMBER|
|CITY|VARCHAR2(21)|
|STATE|VARCHAR2(2)|
|LAT_N|NUMBER|
|LONG_W|NUMBER|

**Solution**--MySQL

```mysql
SELECT CITY, STATE FROM STATION
```
